About This Project
panday.yarsi.ac.id

PANDAY Dashboard is a web-based monitoring system built for YARSI University’s LMS. It helps administrative staff and faculty easily track server status, SSL expiration, suspended accounts, and integration with Spada Indonesia. The dashboard also provides detailed statistics on courses, quizzes, and assignments, complete with Excel export for data analysis. Fully handcrafted with clean code and modern visuals, it delivers both functionality and clarity in one interface. Originally developed during my internship at the Direktorat Pembelajaran Jarak Jauh (DPJJ) in 2020, PANDAY Dashboard was a continuation of a previous intern team's work. Built with Laravel, this project focused on monitoring and visualizing LMS data. After my graduation, the project’s development was continued by Muhammad Arya Kusuma.
